Homelab

My current on budget homelab setup. For the router both network key and admin password has 53 bits entropy, WPS disabled, and MAC filter activated. The raspberry pi hosts a personal VPN service and DNS sinkhole service which to date blocks around 30% of total DNS queries. The NAS is running TrueNAS (formally known as FreeNAS) to act as backup storage for my other computers and media storage. Togheter with these services I can access my network from outside, enjoy DNS blocking (ads, telemetry, malware), compressed VPN tunnel (saves data for mobile connection), and of course know that I have a secure encrypted connection when using public internet.
